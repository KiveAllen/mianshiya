# 面试鸭NEXT前端项目

## 安装依赖

### npm

```bash
npm install
```

### yarn

```bash
yarn
```

### pnpm

```bash
pnpm install
```

### bun

```bash
bun install
```

## 运行项目

### npm

```bash
npm run dev
```

### yarn

```bash
yarn dev
```

### pnpm

```bash
pnpm dev
```

### bun

```bash
bun dev
```
